# License Notes

This directory collects license material and provenance notes for vendored or adapted upstream components used in this repository.

There are two kinds of files here:

- recovered or copied license texts, when an exact license file was available in local history or could be reconstructed reliably,
- provenance notes, when the upstream project and declared license are known but the exact original license file was not preserved in the imported snapshot.

Current contents:

- `PINNacle-MIT.txt`: recovered from local repository history.
- `PINA-MIT.txt`: reconstructed from the current upstream license file.
- `DeepXDE-LGPL-2.1-NOTE.md`: upstream license note and reference.
- `FBPINNs-MIT-NOTE.md`: upstream license note and reference.
- `VPINN-PROVENANCE-NOTE.md`: best-effort provenance note for the experimental VPINN subtree.

For component-by-component mapping, see `THIRD_PARTY.md`.
